# Population-Trends
都道府県別の総人口推移グラフSPA

## API
このアプリケーションでは、(RESAS API)[https://opendata.resas-portal.go.jp/]を使用しています。  
実行の際には、取得したAPI KEYを`.envsample`を参考にして、`.env`ファイルを作成してください。

## Build
ローカルでの実行手順
```
# clone repository
$ git clone https://github.com/rkouhei/Population-Trends.git

# package install
$ npm install

# start
$ npm start
```
